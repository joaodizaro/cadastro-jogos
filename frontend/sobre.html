Sobre o Sistema

GameVault

O GameVault é um sistema de cadastro de jogos de video game desenvolvido como projeto avaliativo da disciplina de Programação II, do curso de Sistemas de Informação (UEMG - Unidade Passos).

O sistema permite cadastrar, listar, editar e excluir jogos, além de consultar estatísticas gerais da coleção através do Dashboard.

Tecnologias utilizadas

  • Front-end: HTML, CSS e JavaScript puro
  • Back-end: Node.js com Express
  • Banco de Dados: SQLite

Funcionalidades

  • Cadastro, edição e exclusão de jogos
  • Listagem com busca e filtro por gênero
  • Página de detalhes de cada jogo
  • Dashboard com estatísticas da coleção

Trabalho individual desenvolvido para fins acadêmicos.
